# AI Resume Analyzer

🔗 Live Demo: https://vikas-chaurasia21-ai-resume-analyze.vercel.app/

🔗 GitHub Repository: https://github.com/vikas-chaurasia21/AI-Resume-Analyzer

## Features
- ATS Score Calculation
- Skills & Missing Skills Detection
- Resume Section Analysis
- Action Word Evaluation
- AI Feedback using Gemini API
- Downloadable Report

## Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
AI: Gemini API
Deployment: Vercel, Railway
